    图片访问不了，可以加入mySQL技术。

    作业练习

    作业一：docker安装Nginx

    • ```shell
      docker search nginx # 去docker hub仓库搜索
      docker pull nginx # download

      后台启动，赋名，使用3344访问默认端口80

      docker run -d –name nginx01 -p:3344:80 nginx
      curl localhost:3344

      1.进入容器

      docker exec -it nginx01 /bin/bash

      2.查看/修改nginx的配置文件 nginx.conf

      whereis nginx
      cd /etc/nginx
      ls

      3.退出

      exit
      docker stop ID
